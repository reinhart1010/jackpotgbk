Ada aja hal yang susah dijelasin, seperti melukis gambar, menyatakan cinta, dan nulis tentang cara kerja game ini.

Kalau kamu tau maksudnya, dan mau perbaiki halaman ini, silahkan buka Issue dan/atau Pull Request di GitHub atau GitLab.

1. Tekan tombolnya.

Setiap pemain hanya punya satu tombol:

Kalau kamu tekan sekali, akan ada ikon (gunting, batu, atau kertas) yang cepat berubah. Tekan lagi untuk memilih ikon yang kamu mau.

Nantinya, ikon baru akan muncul di sebelahnya. Tekan tombol yang sama untuk memilih sampai kepilih 3 item/ikon.

2. Cocokkan 2 item atau lebih.

Kamu harus dapet minimal 2 item (gunting, batu, atau kertas) yang sama, walau urutannya berbeda. Misalnya,

  • (3 gunting), atau
  • (2 batu dan 1 kertas)
Tapi, kalau kamu cuman dapet 1 gunting, 1 batu, dan 1 kertas, kamu akan dapet item (zonk) yang akan masuk ke antrian.

3. Masuk antrian.

Setelah itu, item yang muncul paling banyak tadi masuk antrian. Kalau kamu tadi dapet tiga item sama (misalnya ) itemnya masuk antrian dua kali.

Nanti, semua item yang kamu dapet di antrian bakal ditandingi lawan pemain lainnya.

4. Dapat skor.

Prinsipnya sama:

  • Gunting lawan kertas,
  • Batu lawan gunting
  • Kertas lawan batu.

Setiap pemain dengan dapat skor 0. Buat yang lain, skor dibagikan menurut jumlah pasangan item yang dilawan menang ditambah jumlah dibagi jumlah pemain yang tak dapet .

5. Contoh pembagian skor.

Misalnya, jika ada sepuluh pemain dengan 2 , 3 , 4 , dan 1 , maka:

  • Skor pemain adalah 0,
  • Skor pemain adalah 1 / 4 ( > ) + 2 / (10 - 2) = 0.50,
  • Skor pemain adalah 3 / 3 ( < ) + 2 / (10 - 2) = 1.25,
  • Skor pemain adalah 1 / 1 ( < ) + 2 / (10 - 2) = 1.25,

Tentang, Hak Cipta, dan Lisensi

Kenapa game ini dibuat?

Gue pernah berjanji untuk membuat game ini sejak 2017, bahkan sempat buat model di aplikasi spreadsheet (Microsoft Excel / LibreOffice Calc).

Sekarang, mumpung liburan panjang setelah lulus SMA masih panjang, gue akhirnya mulai buat ini. Mainan ini dibuat menggunakan frontend HTML, CSS, dan JavaScript dan tidak perlu pemasangan/install dulu (langsung jalan).

Rencana berikutnya?

  • Konversi ke PWA (Progressive Web App) untuk Android, iOS, iPadOS, dan Windows 10
  • Google Analytics (dan cara untuk matikan tracking)
  • Lokalisasi (ganti bahasa) ke Bahasa Inggris, dll.
  • Multiplayer (berbasis teknologi IOTA)

Semua ini pastinya tidak bisa dibuat secara cepat. Namun, kalau kamu punya ide kamu bisa kasih saran ke @reinhart1010 di GitHub, GitLab, Instagram, dan Twitter.

Seluruh kode sumber dapat kamu lihat di GitHub atau GitLab.
